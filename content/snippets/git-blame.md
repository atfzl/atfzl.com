# Git Blame

`git blame -w` ignores whitespaces

`git blame -C` checks for code movements between files in a commit.

Can use multiple `-C`:

`git blame -w -C -C -C`

Source: https://blog.gitbutler.com/git-tips-1-theres-a-git-config-for-that/
